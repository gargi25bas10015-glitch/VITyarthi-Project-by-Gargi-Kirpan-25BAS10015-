# VITyarthi-Project-by-Gargi-Kirpan-25BAS10015-
Project title: To=Do List CLI App

Overview : 
	   This project is a simple command-line to-do list application built with Python. It allows users to add, view, and delete tasks efficiently from the terminal.  
           The app is ideal for anyone seeking a lightweight task manager to organize daily activities
	
Fatures :
         Add a new task to the list.
         Display all existing tasks.
         Delete a task by its number.
         Simple and intuitive command-line interface.
         
Technologies/Toold Used :
			 Python 3.x

Steps to install & run:
  			 1.Clone the respository.
  			 2.Navigate to the projecr directory
 			 3.Run python main.py.
   
Instructions for Testing:
  			 Follow the main menu to add , show , or delete tasks.
